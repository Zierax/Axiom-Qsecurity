## Data Integrity

All benchmarks were run with **100% unseen separation** between training and evaluation:

- **IoT Dataset**: 0 raw syscall sequences overlap with the training set — fully out-of-distribution
- **Full Dataset**: Training set (`_dataset_balanced.csv`, 2,974 rec) is a proper subset of the full CSV (5,549 rec); the remaining 2,575 records are unseen by the model

No synthetic or augmented data was used in evaluation.
